# Contributing

Contributions are welcome. To propose a new query or improvement:

1. Fork the repo and create a feature branch
2. Follow the existing query file template (see README)
3. Include MITRE ATT&CK mapping and false-positive notes
4. Submit a pull request with a clear description of the detection logic

Please do not submit queries containing proprietary, employer-specific,
or client-specific logic, thresholds, or identifiers.
